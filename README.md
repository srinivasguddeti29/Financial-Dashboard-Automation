# Financial Dashboard Automation (Company Expenses)

## Project Overview
Financial Dashboard Automation is a Python-based project to track, categorize, and visualize company expenses by department. The tool provides actionable financial insights using data analysis and visualizations.

## Features
- Track company expenses by department
- Perform data analysis to calculate total and departmental expenses
- Visualize expenses with charts for easy understanding
- Simple, beginner-friendly Python project

## Technologies Used
- Python
- Pandas
- Matplotlib

## How to Run
1. Clone the repository.  
2. Ensure `expenses.csv` is in the same folder as `expense_tracker.py`.  
3. Run `expense_tracker.py` in Python.  
4. View analysis and bar chart of expenses by department.
